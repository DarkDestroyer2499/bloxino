Features

Можливості

Everything a classroom needs

Все, що потрібно класу

Built for students and teachers, not just hobbyists.

Створено для учнів та вчителів, а не тільки для ентузіастів.

⚡

Instant Results

No compile-and-upload wait. Changes run on the Arduino board immediately. Students stay in flow.

🌊

Animated Signal Flow

Animated flow shows the exact path your program takes through blocks. Students see what their code does.

📡

Live Pin Values

Every pin's real-time value is visible directly on the blocks while the program runs. No extra tools.

🎨

5 Themed Environments

Space, Ocean, Jungle, Dark, and Soft Rounded. Kids pick their favorite and start coding.

🎓

Teacher Controls

4 difficulty levels. Hide advanced blocks for beginners, unlock everything for advanced students. Free, built-in.

🔌

Any Arduino Board

Uno, Nano, Mega, Leonardo, ESP32. Works with $5 Arduino clones from any shop.

📶

Works Offline

50 MB install, no internet needed. No accounts, no cloud. Works during outages and in classrooms without Wi-Fi.

💻

Real Code Output

Students see the actual Arduino C++ code their blocks generate — a natural bridge to text-based coding.

🚀

Lightweight

Starts in under a second. 50 MB vs 200–800 MB for similar tools. Runs smoothly on old school computers.

⚡

Миттєвий результат

Не потрібно чекати компіляції. Зміни виконуються на платі Arduino одразу. Учні залишаються зосередженими.

🌊

Анімований потік сигналів

Анімація показує точний шлях програми через блоки. Учні візуально розуміють, що робить їхній код.

📡

Значення пінів наживо

Значення кожного піна видно прямо на блоках під час виконання. Без здогадок і додаткових інструментів.

🎨

5 тематичних середовищ

Космос, Океан, Джунглі, Темна та М'яка. Діти обирають улюблену тему і починають програмувати.

🎓

Контроль вчителя

4 рівні складності. Сховайте складні блоки для початківців, відкрийте все для просунутих. Вбудовано безкоштовно.

🔌

Будь-яка плата Arduino

Uno, Nano, Mega, Leonardo, ESP32. Працює з Arduino клонами за $5 з будь-якого магазину.

📶

Працює офлайн

50 МБ, без інтернету. Без акаунтів і хмари. Під час відключень, на старих ПК, у класах без Wi-Fi.

💻

Справжній код

Учні бачать реальний C++ код, який генерують їхні блоки — природний місток до текстового програмування.

🚀

Легкий

Запускається за секунду. 50 МБ замість 200–800 МБ у подібних інструментів. Плавно працює на старих комп'ютерах.
