Why Bloxino?

Чому Bloxino?

Live Execution

No compile/upload cycle. Your blocks execute instantly on the Arduino board via a 58-opcode bytecode VM. Students see results in real time.

Signal Flow Animation

Watch your program flow through the wires between blocks. Students visually understand how code executes — no other tool does this.

Real-Time Debug Overlay

Pin values displayed directly on blocks during execution. See HIGH/LOW states, analog readings, and variable values live.

5 Themed Environments

Space, Ocean, Jungle, Dark, and Soft Rounded — each with unique fonts, colors, and visual effects. Kids love picking their theme.

Teacher Control

4 student levels with per-feature toggles and JSON restriction profiles. Control exactly what students can access. Free, built-in.

Works with Any Arduino

Uno, Nano, Mega, Leonardo, ESP32. No proprietary hardware. Works with $5 Arduino clones from any shop.

Full Offline Support

50 MB installer. Works without internet. No accounts, no cloud dependency. Ideal for classrooms with unreliable connectivity.

Clean Code Generation

Generates readable Arduino C++ with typed variables (int, float, bool, string). Students see real code, not abstracted away.

Native Performance

Built in C++/SFML. Sub-second startup, 60fps, runs on old school PCs. No Electron bloat — 50 MB vs competitors' 200-831 MB.

Живе виконання

Без циклу компіляції/завантаження. Блоки виконуються миттєво на платі Arduino через 58-опкодну байткод VM. Учні бачать результат у реальному часі.

Анімація потоку сигналів

Спостерігайте, як програма тече через з'єднання між блоками. Учні візуально розуміють, як виконується код — жоден інший інструмент цього не робить.

Debug Overlay у реальному часі

Значення пінів відображаються прямо на блоках під час виконання. Бачте стани HIGH/LOW, аналогові значення та змінні наживо.

5 тематичних середовищ

Космос, Океан, Джунглі, Темна та М'яка — кожна з унікальними шрифтами, кольорами та візуальними ефектами. Дітям подобається обирати свою тему.

Контроль вчителя

4 рівні учнів з перемикачами функцій та JSON-профілями обмежень. Контролюйте, що саме доступно учням. Безкоштовно, вбудовано.

Працює з будь-яким Arduino

Uno, Nano, Mega, Leonardo, ESP32. Без пропрієтарного обладнання. Працює з Arduino клонами за $5 з будь-якого магазину.

Повна офлайн-підтримка

Інсталятор 50 МБ. Працює без інтернету. Без акаунтів, без хмарної залежності. Ідеально для класів з нестабільним підключенням.

Чиста генерація коду

Генерує читабельний Arduino C++ з типізованими змінними (int, float, bool, string). Учні бачать справжній код, не абстракцію.

Нативна продуктивність

Побудований на C++/SFML. Запуск за секунду, 60fps, працює на старих шкільних ПК. Без Electron — 50 МБ замість 200-831 МБ у конкурентів.
